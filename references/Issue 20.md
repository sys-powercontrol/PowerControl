# Issue 20: Dashboard do Vendedor

## Descrição
Implementar a página de Dashboard exclusiva para vendedores (usuários com papel `user`).

## Requisitos
- Criar a página `src/pages/SellerDashboard.tsx`.
- Implementar filtro automático para exibir apenas dados do vendedor logado.
- Exibir resumo de vendas do dia e do mês.
- Implementar visualização de comissões acumuladas.
- Adicionar gráfico de barras/linhas mostrando o progresso em relação à meta mensal.
- Reutilizar componentes de cards e gráficos existentes, adaptando o escopo dos dados.
