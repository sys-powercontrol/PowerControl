# Issue 07: Importador de Extrato OFX (`OFXImporter.tsx`)

## Descrição
Implementar funcionalidade para importação de extratos bancários no formato OFX.

## Requisitos
- Criar o componente `src/components/OFXImporter.tsx` na página de Conciliação Bancária.
- Funcionalidade:
    - Upload de arquivo .ofx.
    - Mapeamento de transações do extrato para lançamentos financeiros existentes no sistema.
