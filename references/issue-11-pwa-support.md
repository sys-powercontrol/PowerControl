# Issue 11: Capacidades PWA (Progressive Web App)

## Descrição
Configurar o sistema para funcionar como um PWA, permitindo instalação e uso offline básico.

## Requisitos
- Configuração: Utilizar `vite-plugin-pwa`.
- Requisitos Técnicos:
    - Arquivo `manifest.json` completo com ícones e cores da marca.
    - Implementação de Service Worker para cache de assets estáticos.
    - Sincronização básica em background para permitir operações offline no PDV.
