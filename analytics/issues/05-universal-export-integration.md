# Issue: Integração Universal de Exportação (PDF/Excel)

## Descrição
Adicionar o componente `ExportButton` em todas as telas de listagem de dados para permitir exportação de relatórios.

## Requisitos
- Integrar em: `SalesHistory`, `PurchaseHistory`, `AccountsPayable`, `AccountsReceivable`, `InventoryHistory`.
- Garantir que os filtros aplicados na tela sejam respeitados na exportação.

## Critérios de Aceite
- Todas as tabelas principais do sistema possuem botões funcionais de "Exportar PDF" e "Exportar Excel".
