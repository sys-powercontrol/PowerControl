# Issue: Impressão de Etiquetas de Produtos

## Descrição
Módulo para geração de etiquetas de gôndola ou de produto com código de barras.

## Requisitos
- Componente `LabelPrinter.tsx`.
- Seleção de produtos e quantidades.
- Geração de PDF formatado para impressoras térmicas ou folhas A4.

## Critérios de Aceite
- O sistema gera um PDF com códigos de barras legíveis para os produtos selecionados.
