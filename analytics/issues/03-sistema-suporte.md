# Issue 03: Sistema de Suporte

## Contexto
O sistema de suporte precisa ser mais do que apenas um registro no banco de dados, necessitando de notificações e documentação real.

## Escopo
*   **Páginas:** `Support.tsx`
*   **Componentes:** Formulário de Ticket
*   **Comportamentos:**
    *   **Notificações de Suporte:** Implementar trigger para disparar e-mail ou notificação externa sempre que um novo ticket for aberto.
    *   **Base de Conhecimento:** Substituir o botão visual "Acessar Docs" por um link real ou página de Wiki interna.
